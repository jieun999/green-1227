01 Customized Data Service (CDS, 고객사 맞춤 Data공급)

맞춤 Data상품 공급

고객사 Needs에 부합하는 최적의 Data 상품을 개발하고, 다양한 전송방식을 통해 고객사에 적시 제공하는 신한카드의 Data 서비스 입니다.

주요영역

Macro Data

지역경제, 민간소비, 경제효과, 관광/상권 Data 등

Consumer Data

Profile Data, Micro Segments, 고객유입/이탈 등

Merchant & Seller Data

온/오프라인 사업자 Data

E-Commerce Data

e커머스 카테고리/브랜드/상품 Data

Market Data

Market Share, 고객유입/이탈, Brand Performance Data 등
